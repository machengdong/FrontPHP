# FrontPHP

    FrontPHP 是一款轻量级的PHP框架，作者抱着重复造轮子的想法，在尽量精简代码量的前提下，让其继续支持一般WEB框架应有的功能。
     


*  基础MVC编码构架的支持

*  最小化配置
    
*  多数据库场景的灵活使用
    
*  遵循PSR代码规范
    
*  部署方便，内置Swoole作为HTTP服务器
